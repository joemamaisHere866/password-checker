# 🔐 Password Strength Checker

A simple Python tool to check password strength and security created in vscode

## Features
- Checks password length
- Detects uppercase/lowercase letters
- Verifies numbers and special characters
- Identifies common passwords
- Provides strength score (Weak/Medium/Strong)
- Gives improvement suggestions

## How to Use

### Prerequisites
- Python 3.x installed

### Run the Program
```bash
python password_checker.py
